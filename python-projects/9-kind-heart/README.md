# Оценка эффективности деятельности благотворительного фонда "АиФ. Доброе сердце"

## Описание проекта

**Объект исследования**\
Благотворительный фонд «АиФ. Доброе сердце» работает с $2005$ года по инициативе газеты «Аргументы и факты».\
За почти $20$ лет работы более $10$ тысяч подопечных со сложными диагнозами получили помощь оплатой лечения, лекарств и медобслуживания.

**Цель проекта**\
Фонд желал лучше узнать своих благотворителей. Всё исследование посвящено этому.\
Таким образом фонд сможет эффективно выстроить взаимодействие с донорами.

**Для достижения поставленной цели были решены следующие задачи:**
1. Охарактеризован портрет типичного донора.
2. Узнали, как благотворители ведут свою активность в фонде.
3. Выяснили, какова статистика по каналам привлечения благотворителей.
4. Оценили эффективность маркетинга.
5. Провели RFM-анализ.
6. Провели когортный анализ.

**Описание файлов:** \
`kind_heart.ipynb` — тетрадка Jupyter с работой.

**Email-маркетинг**\
Из $891$ $420$ отправленных сообщений было открыто всего лишь $145$ $888$ ($16.4$%) из них, а кликнули по ссылкам внутри писем всего лишь $16$ $595$ раз ($1.9$%).\
Иначе говоря, конверсия отправки письма в клик по ссылке (т.е. в переход на сайт фонда) равна $1.9$%.\
Считаются только уникальные случаи. Если благотворитель $50$ раз открыл письмо и $10$ раз кликнул по ссылке, то будет считаться, что он открыл и кликнул только по одному разу.

4. Для заказчика совершена выгрузка данных с исходной таблицей RFM-анализа, сводной таблицей со статистикой, таблицей с идентификаторами доноров по сегментам.

**Когортный анализ**\
Рассчитаны метрики:  `Retention Rate`, `Cumulative Revenue`, `LTV`, `AOV`:
* `Retention Rate` на $30$-й день для жертвовавших благотворителей — $1$% для неисторических и $1.53$% для исторических.
* `Cumulative Revenue`. К $30$-му дню жизни все неисторические принесли $10$ $182$ $810$ ₽, а исторические — $1$ $988$ $750$ ₽.\
Неисторические превосходят в $5.12$ раза.
* `LifeTime Value`. К $30$-му дню жизни средний неисторический приносит $769$ ₽, а исторический — $890$ ₽.\
Исторические превосходят в $1.15$ раза.
* `Average Order Value`. $30$-му дню жизни средний чек пожертвования неисторического равен $430$ ₽, а исторического — $174$ ₽.\
Неисторические превосходят в $2.47$ раза.
